How the web works

Essentially, the World Wide Web is a network that connects computers with servers.
The computers "talk" to servers by sending something called HTTP requests(Hypertext Transfer Protocol requests).

HTML

HTML or Hyypertext Markup Language is the standard markup language used to create web pages.
It is written in the form of HTML elements consisting of tags enclosed in angle brackets.

Tags and Elements

An HTML tag is always contained within angled brakets. Most tags have an opening tag and a closing tag. Void tags do not require closing tags.
An HTML element refers to everything within a set of opening and closing tags.

Inline vs. Block elements

Inline elements format everything in a single line. Examples are: br (break), em (embolden), and img (image).
Block elements formats the contents within an invisible textbox. Examples are: p (paragraph) and div (divide).
